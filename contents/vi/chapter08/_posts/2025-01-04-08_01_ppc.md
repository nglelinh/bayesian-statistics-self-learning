## Mục tiêu Học tập

Sau khi hoàn thành bài học này, bạn sẽ hiểu về **Posterior Predictive Checks (PPC)** - công cụ quan trọng nhất để đánh giá model quality trong Bayesian statistics. Bạn sẽ học cách generate predictive data, so sánh với observed data, và identify model misspecification. Đây là bước thiết yếu trong Bayesian workflow - không bao giờ tin model mà không check!

## Giới thiệu: Models Are Always Wrong

> **"All models are wrong, but some are useful."** - George Box

**Câu hỏi quan trọng**: Model của chúng ta có **useful** không?

**PPC philosophy**: Nếu model tốt, nó phải **generate data giống observed data**.

## 1. Posterior Predictive Distribution

### 1.1. Mathematical Definition

**Posterior Predictive Distribution**:
$$
p(\tilde{y} | y) = \int p(\tilde{y} | \theta) p(\theta | y) d\theta
$$

Trong đó:
- $$y$$: Observed data
- $$\tilde{y}$$: Predicted/replicated data
- $$\theta$$: Parameters
- $$p(\theta \mid y)$$: Posterior

**Intuition**: 
1. Sample $$\theta$$ from posterior
2. Generate $$\tilde{y}$$ from likelihood $$p(\tilde{y} \mid \theta)$$
3. Repeat many times

```python
import numpy as np
import matplotlib.pyplot as plt
import pymc as pm
import arviz as az

# Generate data
np.random.seed(42)
n = 50
x = np.random.uniform(0, 10, n)
y_true = 2 + 0.5*x + np.random.normal(0, 1, n)

# Fit model
x_z = (x - x.mean()) / x.std()
y_z = (y_true - y_true.mean()) / y_true.std()

with pm.Model() as model:
    alpha = pm.Normal('alpha', 0, 1)
    beta = pm.Normal('beta', 0, 1)
    sigma = pm.HalfNormal('sigma', 1)
    
    mu = alpha + beta * x_z
    y_obs = pm.Normal('y_obs', mu=mu, sigma=sigma, observed=y_z)
    
    # Sample posterior
    trace = pm.sample(1000, tune=500, chains=2, random_seed=42,
                     return_inferencedata=True, progressbar=False)
    
    # Posterior predictive
    ppc = pm.sample_posterior_predictive(trace, random_seed=42)

# Visualize process
fig, axes = plt.subplots(1, 3, figsize=(18, 5))

# Step 1: Posterior samples
alpha_samples = trace.posterior['alpha'].values.flatten()
beta_samples = trace.posterior['beta'].values.flatten()

axes[0].scatter(alpha_samples[:100], beta_samples[:100], s=30, alpha=0.5,
               edgecolors='black')
axes[0].set_xlabel('α', fontsize=12, fontweight='bold')
axes[0].set_ylabel('β', fontsize=12, fontweight='bold')
axes[0].set_title('STEP 1: Sample θ from Posterior\np(θ \mid y)',
                 fontsize=14, fontweight='bold')
axes[0].grid(alpha=0.3)

# Step 2: Generate predictions
y_pred_samples = ppc.posterior_predictive['y_obs'].values.reshape(-1, n)

for i in range(20):
    axes[1].scatter(x, y_pred_samples[i], s=20, alpha=0.3)
axes[1].scatter(x, y_z, s=60, color='red', edgecolors='black',
               label='Observed', zorder=5)
axes[1].set_xlabel('x', fontsize=12, fontweight='bold')
axes[1].set_ylabel('y', fontsize=12, fontweight='bold')
axes[1].set_title('STEP 2: Generate ỹ from p(ỹ \mid θ)\n20 replications',
                 fontsize=14, fontweight='bold')
axes[1].legend(fontsize=11)
axes[1].grid(alpha=0.3)

# Step 3: Compare distributions
axes[2].hist(y_z, bins=20, alpha=0.6, density=True, edgecolor='black',
            label='Observed', color='red')
for i in range(min(50, y_pred_samples.shape[0])):
    axes[2].hist(y_pred_samples[i], bins=20, alpha=0.02, density=True,
                color='blue')
axes[2].set_xlabel('y', fontsize=12, fontweight='bold')
axes[2].set_ylabel('Density', fontsize=12, fontweight='bold')
axes[2].set_title('STEP 3: Compare Distributions\nObserved vs Predicted',
                 fontsize=14, fontweight='bold')
axes[2].legend(fontsize=11)
axes[2].grid(alpha=0.3, axis='y')

plt.tight_layout()
plt.show()

print("=" * 70)
print("POSTERIOR PREDICTIVE CHECK PROCESS")
print("=" * 70)
print("\n1. Sample parameters from posterior: p(θ \mid y)")
print("2. For each θ, generate data: ỹ ~ p(ỹ \mid θ)")
print("3. Compare ỹ with observed y")
print("\n→ If model good: ỹ should look like y!")
print("=" * 70)
```

### 2.2. Test Statistics

Check specific features of data:

```python
# Compute test statistics
def compute_statistics(data):
    return {
        'mean': data.mean(),
        'std': data.std(),
        'min': data.min(),
        'max': data.max(),
        'q25': np.percentile(data, 25),
        'q75': np.percentile(data, 75)
    }

# Observed statistics
obs_stats = compute_statistics(y_z)

# Predicted statistics
pred_stats = {key: [] for key in obs_stats.keys()}
for i in range(y_pred_samples.shape[0]):
    stats = compute_statistics(y_pred_samples[i])
    for key in stats:
        pred_stats[key].append(stats[key])

# Visualize
fig, axes = plt.subplots(2, 3, figsize=(18, 10))
axes = axes.ravel()

stat_names = list(obs_stats.keys())
for idx, stat_name in enumerate(stat_names):
    axes[idx].hist(pred_stats[stat_name], bins=30, alpha=0.7,
                  edgecolor='black', density=True, label='Predicted')
    axes[idx].axvline(obs_stats[stat_name], color='red', linewidth=3,
                     label='Observed')
    axes[idx].set_xlabel(f'{stat_name.upper()}', fontsize=12, fontweight='bold')
    axes[idx].set_ylabel('Density', fontsize=12, fontweight='bold')
    axes[idx].set_title(f'PPC: {stat_name.upper()}\n' +
                       f'Obs = {obs_stats[stat_name]:.2f}',
                       fontsize=13, fontweight='bold')
    axes[idx].legend(fontsize=10)
    axes[idx].grid(alpha=0.3, axis='y')

plt.tight_layout()
plt.show()

print("\n" + "=" * 70)
print("TEST STATISTICS PPC")
print("=" * 70)
for stat_name in stat_names:
    pred_vals = pred_stats[stat_name]
    obs_val = obs_stats[stat_name]
    p_value = np.mean(np.array(pred_vals) >= obs_val)
    
    print(f"\n{stat_name.upper()}:")
    print(f"  Observed: {obs_val:.3f}")
    print(f"  Predicted mean: {np.mean(pred_vals):.3f}")
    print(f"  Bayesian p-value: {p_value:.3f}")
    
    if p_value < 0.05 or p_value > 0.95:
        print(f"  → Potential problem!")
print("=" * 70)
```

## 3. Detecting Model Misspecification

### 3.1. Example: Wrong Likelihood

```python
# Generate data with outliers
np.random.seed(42)
n = 100
x_out = np.random.uniform(0, 10, n)
y_out = 2 + 0.5*x_out + np.random.normal(0, 1, n)
# Add outliers
y_out[::10] += np.random.choice([-10, 10], size=10)

x_out_z = (x_out - x_out.mean()) / x_out.std()
y_out_z = (y_out - y_out.mean()) / y_out.std()

# Fit Normal model (WRONG for outliers!)
with pm.Model() as model_normal:
    alpha = pm.Normal('alpha', 0, 1)
    beta = pm.Normal('beta', 0, 1)
    sigma = pm.HalfNormal('sigma', 1)
    
    mu = alpha + beta * x_out_z
    y_obs = pm.Normal('y_obs', mu=mu, sigma=sigma, observed=y_out_z)
    
    trace_normal = pm.sample(1000, tune=500, chains=2, random_seed=42,
                            return_inferencedata=True, progressbar=False)
    ppc_normal = pm.sample_posterior_predictive(trace_normal, random_seed=42)

# Fit Student-t model (BETTER for outliers!)
with pm.Model() as model_t:
    alpha = pm.Normal('alpha', 0, 1)
    beta = pm.Normal('beta', 0, 1)
    sigma = pm.HalfNormal('sigma', 1)
    nu = pm.Gamma('nu', 2, 0.1)  # Degrees of freedom
    
    mu = alpha + beta * x_out_z
    y_obs = pm.StudentT('y_obs', nu=nu, mu=mu, sigma=sigma, observed=y_out_z)
    
    trace_t = pm.sample(1000, tune=500, chains=2, random_seed=42,
                       return_inferencedata=True, progressbar=False)
    ppc_t = pm.sample_posterior_predictive(trace_t, random_seed=42)

# Compare PPCs
fig, axes = plt.subplots(1, 2, figsize=(16, 6))

# Normal model
y_pred_normal = ppc_normal.posterior_predictive['y_obs'].values.reshape(-1, n)
axes[0].hist(y_out_z, bins=30, alpha=0.6, density=True, edgecolor='black',
            label='Observed', color='red')
for i in range(min(50, y_pred_normal.shape[0])):
    axes[0].hist(y_pred_normal[i], bins=30, alpha=0.02, density=True,
                color='blue')
axes[0].set_xlabel('y', fontsize=12, fontweight='bold')
axes[0].set_ylabel('Density', fontsize=12, fontweight='bold')
axes[0].set_title('NORMAL LIKELIHOOD\nMisses outliers!',
                 fontsize=14, fontweight='bold', color='red')
axes[0].legend(fontsize=11)
axes[0].grid(alpha=0.3, axis='y')

# Student-t model
y_pred_t = ppc_t.posterior_predictive['y_obs'].values.reshape(-1, n)
axes[1].hist(y_out_z, bins=30, alpha=0.6, density=True, edgecolor='black',
            label='Observed', color='red')
for i in range(min(50, y_pred_t.shape[0])):
    axes[1].hist(y_pred_t[i], bins=30, alpha=0.02, density=True,
                color='green')
axes[1].set_xlabel('y', fontsize=12, fontweight='bold')
axes[1].set_ylabel('Density', fontsize=12, fontweight='bold')
axes[1].set_title('STUDENT-T LIKELIHOOD\nCaptures outliers!',
                 fontsize=14, fontweight='bold', color='green')
axes[1].legend(fontsize=11)
axes[1].grid(alpha=0.3, axis='y')

plt.tight_layout()
plt.show()

print("\n" + "=" * 70)
print("MODEL COMPARISON via PPC")
print("=" * 70)
print("\nNormal model:")
print("  → Cannot generate extreme values")
print("  → Underestimates tails")

print("\nStudent-t model:")
print("  → Heavy tails")
print("  → Captures outliers better")

print("\n→ PPC reveals model misspecification!")
print("=" * 70)
```

## 4. Bayesian p-values

**Bayesian p-value**: Proportion of predicted data "more extreme" than observed.

$$
p_B = P(T(\tilde{y}) \geq T(y) | y)
$$

where $$T(\cdot)$$ is a test statistic.

**Interpretation**:
- p ≈ 0.5: Good (observed typical)
- p < 0.05 or p > 0.95: Potential problem

```python
# Compute Bayesian p-values for multiple statistics
def bayesian_p_value(observed_stat, predicted_stats):
    return np.mean(predicted_stats >= observed_stat)

# Test statistics
test_stats = {
    'Max': (np.max, 'Maximum value'),
    'Min': (np.min, 'Minimum value'),
    'Range': (lambda x: x.max() - x.min(), 'Range'),
    'IQR': (lambda x: np.percentile(x, 75) - np.percentile(x, 25), 'IQR'),
    'Skewness': (lambda x: ((x - x.mean())**3).mean() / x.std()**3, 'Skewness')
}

print("\n" + "=" * 70)
print("BAYESIAN P-VALUES")
print("=" * 70)

for stat_name, (stat_func, description) in test_stats.items():
    obs_stat = stat_func(y_z)
    pred_stats = [stat_func(y_pred_samples[i]) for i in range(y_pred_samples.shape[0])]
    p_val = bayesian_p_value(obs_stat, np.array(pred_stats))
    
    print(f"\n{stat_name} ({description}):")
    print(f"  Observed: {obs_stat:.3f}")
    print(f"  Bayesian p-value: {p_val:.3f}")
    
    if p_val < 0.05:
        print(f"  → Observed unusually LOW")
    elif p_val > 0.95:
        print(f"  → Observed unusually HIGH")
    else:
        print(f"  → Looks good ✓")

print("=" * 70)
```

## Tóm tắt

Posterior Predictive Checks (PPC):

- **Philosophy**: Good model generates data similar to observed
- **Process**: Sample θ → Generate ỹ → Compare with y
- **Visual**: Most important - look at distributions
- **Test statistics**: Check specific features (mean, variance, extremes)
- **Bayesian p-values**: Quantify discrepancy

**Key insight**: PPC is **model criticism**, not model selection. Use it to find problems, then fix model!

Bài tiếp theo: **Information Criteria** (WAIC, LOO) cho model comparison.
