# How to Contribute

We welcome contributions from the community! Whether you want to fix a typo, add an example, or contribute a whole new chapter, your help is appreciated.

## Ways to Contribute

- **Fix errors**: Found a typo or mistake? Submit a fix!
- **Add examples**: Share practical examples or exercises
- **Improve explanations**: Make concepts clearer
- **Translate content**: Help make the course available in more languages
- **Add resources**: Suggest additional reading materials or tools

## Getting Started

1. **Fork the repository** on GitHub
2. **Clone your fork** to your local machine
3. **Create a new branch** for your changes
4. **Make your changes** following our style guide
5. **Test your changes** locally
6. **Submit a pull request** with a clear description

## Style Guidelines

- Use clear, concise language
- Follow the existing formatting conventions
- Include examples where appropriate
- Test all code snippets
- Use proper LaTeX formatting for mathematical expressions

## Questions?

If you have questions about contributing, please:
- Check our [Contributing Guide]({{ site.baseurl }}/contribution/how_to_contribute/)
- Open an issue on GitHub
- Contact the course maintainers

Thank you for helping improve this course!
