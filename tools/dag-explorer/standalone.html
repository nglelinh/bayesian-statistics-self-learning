📘 Hướng dẫn Sử dụng Interactive DAG Explorer

🎯 Mục đích

Công cụ này giúp bạn:

  • Vẽ DAG (Directed Acyclic Graph) trực quan
  • Xác định adjustment sets tự động
  • Phát hiện backdoor paths
  • Hiểu confounding, mediation, và collider bias

🖱️ Cách sử dụng

  • Thêm node: Nhập tên và click "Add Node"
  • Thêm edge: Chọn "From" và "To", click "Add Edge"
  • Di chuyển node: Kéo thả (drag & drop) node trên canvas
  • Xóa node: Right-click vào node hoặc chọn node và nhấn Delete/Backspace
  • Xóa edge: Click vào edge (mũi tên)
  • Undo/Redo: Ctrl+Z / Ctrl+Y hoặc dùng buttons
  • Chọn Treatment/Outcome: Dùng dropdowns bên phải để phân tích causal
  • Load ví dụ: Click "Example DAGs" để xem mẫu

📊 Ví dụ DAG có sẵn

  • Fork: Z → X, Z → Y (Z is confounder)
  • Chain: X → M → Y (M is mediator)
  • Collider: X → C ← Y (C is collider)
  • Simpson's Paradox: Confounding example
